# Complie
```shell
make
```
If is not working, please try:
```shell
clang -m64 -O3 -Wall -lpthread -o thread_socket thread_socket.c
```
# Usage
```shell
./thread_socket -h
Usage of ./thread_sock:
        -p      <port for running>
        -l      show running log
        -d      start daemon service
        -h      show this message
```
