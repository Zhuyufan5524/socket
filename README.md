# Complie
```shell
clang -m64 -O3 -Wall -lpthread -o thread_sock thread_sock.c
```
# Usage
```shell
./thread_sock -h
Usage of ./thread_sock:
        -p      <port for running>
        -l      show running log
        -d      start daemon service
        -h      show this message
```
